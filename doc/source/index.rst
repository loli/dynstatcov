DynStatCov
==========

:Release: |release|
:Date: |today|

DynStatCov - Cython Library for fast dynamic statistical co-variance update.

This is a speed-optimized implementation to be used in other algorithms. See :doc:`usage/introduction` for what it does, :doc:`usage/usage` for how to use it and :doc:`usage/background` if you are interested in the inner workings. The whole API documentation can be found under the heading References below.
